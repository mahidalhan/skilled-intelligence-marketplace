# Analyze Architecture

Introspect and analyze software architectures using SpaceX's 5-step engineering methodology.

Uses the @architecture-introspector skill to identify unnecessary complexity, validate architectural decisions, and recommend improvements.

## Process

1. Question every requirement
2. Delete unnecessary parts
3. Simplify what remains
4. Accelerate cycle time
5. Automate last

## Usage

Run this command then describe the architecture to analyze or point to the codebase area.
