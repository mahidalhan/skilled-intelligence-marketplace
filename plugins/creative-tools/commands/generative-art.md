# Generative Art

Create gallery-quality algorithmic art using p5.js with interactive parameter exploration.

Uses the @algorithmic-art skill to generate computational art through:
- Flow fields and particle systems
- Noise functions and emergent behavior
- Seeded randomness for reproducibility
- Interactive parameter controls

## Process

1. Creates an algorithmic philosophy (the conceptual foundation)
2. Expresses it through p5.js code
3. Outputs a self-contained HTML artifact with controls

## Usage

Describe the aesthetic or concept:
- `/generative-art organic turbulence`
- `/generative-art geometric harmony`
- `/generative-art recursive patterns`
