# Create Diagram

Generate exceptional Mermaid diagrams for ERDs, flowcharts, sequence diagrams, and system visualizations.

Uses the @mermaid-diagrams skill to create clear, refined diagrams with proper visual hierarchy.

## Diagram Types

- `erDiagram` - Data models with relationships
- `flowchart` - Processes and decision flows
- `sequenceDiagram` - System interactions
- `stateDiagram-v2` - State machines and lifecycles
- `classDiagram` - OOP structures
- `C4Context` - Architecture overviews

## Usage

Describe what you want to visualize:
- `/diagram the authentication flow`
- `/diagram user registration ERD`
- `/diagram API request sequence`
