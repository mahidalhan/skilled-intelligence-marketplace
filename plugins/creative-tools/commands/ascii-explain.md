# ASCII Explain

Explain code concepts and systems using ASCII art diagrams and visualizations.

Uses the @ascii-art-explainer skill to create text-based diagrams that work in any terminal or text environment.

## Usage

Provide the concept to visualize:
- `/ascii-explain how a hash table works`
- `/ascii-explain binary tree traversal`
- `/ascii-explain request-response cycle`
