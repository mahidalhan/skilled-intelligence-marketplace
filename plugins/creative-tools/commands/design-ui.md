# Design UI

Build memorable, production-grade frontend interfaces that avoid generic AI aesthetics.

Uses the @frontend-design skill to create bold, distinctive designs with exceptional attention to typography, color, motion, and spatial composition.

## Design Approach

- Bold aesthetic direction (minimalist, maximalist, retro-futuristic, etc.)
- Distinctive typography choices
- Intentional color palettes
- Motion and micro-interactions
- Unexpected layouts and compositions

## Usage

Describe the interface you need:
- `/design-ui landing page for a SaaS product`
- `/design-ui dashboard component`
- `/design-ui pricing page`
