# Create Skill

Create exceptional, deterministic Claude skills that guide AI behavior with precision.

Uses the @skill-creator skill to generate structured skill files following the canonical format.

## Output Format

- YAML frontmatter with name, description, license
- Thinking section with strategic questions
- CRITICAL callout with the core principle
- Guidelines section with 5 focus areas
- NEVER section listing anti-patterns
- ~40-45 lines total

## Usage

Describe the skill you want to create:
- `/create-skill for code review`
- `/create-skill for writing tests`
- `/create-skill for documentation`
