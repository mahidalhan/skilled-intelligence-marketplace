# XML Context

Structure prompts and context using XML tags for optimal LLM comprehension.

Uses the @xml-context-engineering skill to transform unstructured content into well-tagged XML that creates clear attention anchors.

## Principles

- Semantic tag naming (domain-native names)
- Flat-first structure (siblings over nesting)
- Progressive disclosure (important content first)
- Minimum viable structure

## Usage

Provide the content to structure:
- `/xml-context this system prompt`
- `/xml-context these instructions for an agent`
- `/xml-context this multi-part specification`
